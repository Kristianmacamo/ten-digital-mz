# TEN Digital MZ - Site Oficial

Site profissional da TEN Digital MZ - Agência de Soluções Digitais.

## 🚀 Características

- ✅ Design moderno e responsivo
- ✅ Navegação suave (smooth scroll)
- ✅ Menu hamburger para mobile
- ✅ Animações interativas
- ✅ Formulário de contato funcional
- ✅ Seções bem estruturadas
- ✅ Performance otimizada
- ✅ SEO friendly

## 📁 Estrutura do Projeto

```
ten-digital-mz/
├── index.html      # Página principal
├── styles.css      # Estilos globais
├── script.js       # Funcionalidades JavaScript
└── README.md       # Este arquivo
```

## 🎨 Seções do Site

1. **Navegação** - Menu principal com links para todas as seções
2. **Hero** - Seção inicial chamativa com call-to-action
3. **Sobre** - Informações sobre a empresa com estatísticas
4. **Serviços** - Cards com os principais serviços oferecidos
5. **Portfolio** - Showcase de projetos realizados
6. **Contato** - Formulário de contato e informações
7. **Footer** - Rodapé com links sociais

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (Grid, Flexbox, Animações)
- JavaScript Vanilla
- Font Awesome Icons

## 📱 Responsividade

O site é totalmente responsivo e funciona perfeitamente em:
- Desktop (1920px+)
- Tablet (768px - 1023px)
- Mobile (até 767px)

## 🚀 Como Usar

### Localmente

1. Clone ou baixe o repositório
2. Abra `index.html` em seu navegador
3. Ou use um servidor local:
   ```bash
   python -m http.server 8000
   # Acesse http://localhost:8000
   ```

### Deploy com GitHub Pages

1. Vá para Configurações do repositório
2. Procure por "Pages" na barra lateral
3. Selecione "main" como branch
4. Seu site estará disponível em: `https://Kristianmacamo.github.io/ten-digital-mz`

## 🎯 Funcionalidades JavaScript

- **Menu Hamburger** - Menu responsivo para mobile
- **Smooth Scroll** - Navegação suave entre seções
- **Scroll Animations** - Animações ao scroll
- **Form Validation** - Validação do formulário de contato
- **Scroll to Top** - Botão flutuante para voltar ao topo

Para personalizar o site, edite:
